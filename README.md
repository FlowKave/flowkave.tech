# FlowKave Site

Professional MVP website for FlowKave — a managed AI-agent workflow business.

## Current scope

- Three-language landing page: English, Persian, Swedish
- Coffee equipment seller demo case study
- Demo client portal preview under `/client/coffee-demo`
- Contact email: `flowkave@gmail.com`
- Built with Next.js + Tailwind CSS

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Recommended deployment

- Push this repo to GitHub.
- Deploy via Vercel.
- Connect `flowkave.tech` to the Vercel project.
- Later add Cloudflare email routing: `hello@flowkave.tech -> flowkave@gmail.com`.
